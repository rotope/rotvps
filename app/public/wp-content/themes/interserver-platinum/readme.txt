== Description ==

Interserver Platinum is a multipurpose theme for various type of businesses. It is highly custoomizable and easy to use.

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

============= Demo Import Instructions =============
1. After activation of the theme, install and activate all the recommended plugins.
2. You can import demo data by the importing following files to your wordpress website
- http://stagingurl.website/ip-content.xml
- http://stagingurl.website/ip-widgets.wie
- http://stagingurl.website/ip-customizer.dat

After that set "Home" as Home page and "Blog" as PostsPage from Appearance > Customizer > Homepage Settings ans set Main Menu as primary menu and Footer Menu as footer menu from Appearance > Menus
